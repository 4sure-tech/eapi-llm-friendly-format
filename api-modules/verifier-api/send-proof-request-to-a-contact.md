# Send Proof Request to A Contact

The _**Send Proof Request to Contact API**_ enables the LOB to send a proof request to a contact, or peer. The API delivers the proof request to the contact using the protocol supported by the contact. The protocol of the recipient is known from the public DID document of the recipient. \
