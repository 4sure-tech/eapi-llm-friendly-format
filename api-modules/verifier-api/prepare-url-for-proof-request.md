# Prepare URL for Proof Request

The _**Prepare URL for Proof Request API**_ enables the LOB to receive a URL-based proof request, which can then be sent as an out-of-band proof request. This URL will be created based on the protocol specified. The client can determine how the URL is sent to the intended recipient of the proof request. \
